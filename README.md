
<h1>LoginRegistration</h1>



<h2>Description</h2>
Basic login and registration system implemented using ASP.Net core and Razor pages with SQL server for database.
(Detailed documentation available as .docx)

<h2>Languages and Utilities Used</h2>

- <b> ASP.NET Core Razor Pages </b>
- <b> Identity on ASP.NET Core: </b>
- <b> Bootstrap for ccs framework </b>
- <b> SQL server for database </b>
- <b> Entity Framework </b>
